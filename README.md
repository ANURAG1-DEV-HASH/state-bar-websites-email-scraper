# State Bar Websites Email Scraper
This scraper extracts names and email addresses from state bar websites across the US. It is designed to help users gather public contact data from legal professional directories to streamline legal networking or research tasks. It uses web crawling techniques with a focus on accuracy and detail, employing Scrapy for optimal performance.

## Introduction
This project provides a web scraping tool to extract names and email addresses from various state bar association websites across the United States. The primary goal is to gather publicly available contact information for legal professionals. This tool is ideal for researchers, legal professionals, and businesses seeking to connect with members of the legal community.

### Importance of Scraping State Bar Websites
- Helps researchers and legal professionals find relevant contacts in the legal field.
- Automates the tedious process of manually gathering contact information.
- Provides an efficient and scalable method for acquiring public data from multiple state bar websites.
- Ensures accuracy by focusing on extracting only the most relevant data: names and emails.

## Features
| Feature               | Description                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| Multi-State Support    | Supports scraping from various state bar websites across the United States.  |
| Accuracy Focused       | Extracts only the most relevant contact details (names and emails).         |
| Scrapy Powered         | Utilizes Scrapy framework for efficient and reliable web crawling.          |
| Data Output in JSON    | Results are exported in a structured JSON format for easy use and analysis. |

---

## What Data This Scraper Extracts
| Field Name    | Field Description                                  |
|---------------|-----------------------------------------------------|
| name          | Full name of the legal professional.               |
| email         | Email address of the legal professional.           |
